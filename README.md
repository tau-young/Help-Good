>
> 在疫情期间，各个小区居民发挥互助精神，进行物品交换，互通有无。请你编写一个物品交换软件

>
> 你实现的程序可以采用命令行方式使用，但是鼓励提供GUI

> 程序实现完毕后，发布在Github上
